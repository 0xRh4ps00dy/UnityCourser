There are two errors in the line of code below. What are the errors?

transform.Rotate(0, 0.25, 0) >

Select the two correct answers.

  1. The line of code should end with a semicolon (;) instead of a greater than sign (>).

  2. The number 0.25 should have an "f" suffix to explicitly indicate it is a float (0.25f).

  3. The Rotate method does not accept three arguments.

  4. The transform object needs to be capitalized (Transform) to be correctly referenced in Unity.

Select the three correct answers.
